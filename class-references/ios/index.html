Overview

Smooch lets you have rich conversations with your users, right in your app or on your website.

The Smooch class initializes the SDK, and shows the conversation.

An SKTSettings object is passed to the initWithSettings: method to apply configuration options such as the app token, accent color, and notification display time.

The SKTUser class is used to attach custom user properties to each conversation.

The SKTConversation class allows you to send and retrieve messages, as well as set a delegate to respond to important events.

The SKTMessage class provides information about each message in the conversation, including it’s text, the author’s name, and more.

Read our installation guide to get started.
