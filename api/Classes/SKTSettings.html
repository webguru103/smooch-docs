The app token corresponding to your application.

The accent color for the conversation screen.

The status bar style to use on the conversation screen.

enableLocalNotification

A Boolean property that indicates whether to show a local OS notification that brings your user into the conversation page once tapped.

@property BOOL enableLocalNotification

Discussion

The local OS notification is only shown if a user searched for help and left the app within 20 seconds, but without reading any KB articles or attempting to start a conversation.

The default value is YES.

knowledgeBaseURL

The base URL of your Zendesk knowledge base, to be used in constructing the search endpoint.

@property (nonatomic, copy) NSString *knowledgeBaseURL

Discussion

This value may only be set once. If the knowledgeBaseURL is not specified at init time, search is disabled.

The URL must be fully qualified, including http or https (ex: “https://supportkit.zendesk.com”).

The default value is nil.

excludeSearchResultsIf:categories:sections:

Sets the filtering policy applied to user search results based on the given filter mode.

- (void)excludeSearchResultsIf:(id)filterMode categories:(id)categories sections:(id)sections

Parameters

filterMode

The filter mode to use.

categories

Array of category ids on which to filter search results. Can be objects of type NSString or NSNumber.

sections

Array of section ids on which to filter search results. Can be objects of type NSString or NSNumber.

Discussion

Filtering may only be configured once, and configuration must be done at init time or no filtering will be applied.

Filtering by category id is only possible for Zendesk instances that are using HelpCenter.
